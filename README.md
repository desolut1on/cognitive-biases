# Когнитивные искажения — обучающий проект

Интерактивное веб-приложение для изучения когнитивных искажений: теория, примеры, тесты и отслеживание прогресса.

---

## О проекте

Проект создан для знакомства с основными когнитивными искажениями. Он содержит:

- список модулей с кратким описанием и временем изучения,
- страницу модуля с теорией, примерами и списком искажений,
- тест для проверки знаний (пошаговый, с подсчётом правильных ответов),
- страницу результатов с разбором ошибок и правильными ответами,
- прогресс-бар и счётчик пройденных модулей,
- сохранение прогресса в `localStorage`.

---

## Технологии

- React
- React Router (навигация между страницами)
- Tailwind CSS (стилизация и адаптив)
- `localStorage` (хранение прогресса)
